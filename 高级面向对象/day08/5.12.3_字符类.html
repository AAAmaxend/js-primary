正则表达式中的特殊字符：字符类表示有一系列字符可供选择，只要匹配其中一个就可以了。所有可供选择的字符都放在方括号内。

1. [] 方括号： /[abc]/.test('andy') // true ,后面的字符串只要包含 abc 中任意一个字符，都返回 true

2. [-] 方括号内部范围符- ： /^[a-z]$/.test(c') // true 这里表示 a 到 z 26个英文字母都可以。注意和边界符 ^ 区别，边界符写到方括号外面。

3. ^[] 方括号内部取反符^ : /[^abc]/.test('andy') // false,方括号内部加上 ^ 表示取反，只要包含方括号内的字符，都返回 false 。

4. 字符组合 : /[a-z1-9]/.test('andy') // true,，这里表示包含 a 到 z 的26个英文字母和 1 到 9 的数字都可以。
